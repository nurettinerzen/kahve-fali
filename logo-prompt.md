# Çiftler Düellosu — Logo Üretim Prompt'ları

Marka özeti (prompt'ların dayandığı zemin):
- Ürün: çiftler için premium soru-cevap oyunu. Eşler soruları cevaplar, birbirinin
  cevabını tahmin eder. Sıcak, mahrem, zarif — şirin/çocuksu DEĞİL, klişe-romantik DEĞİL.
- Palet: fildişi #F6F2EA zemin, kömür #241D13, bronz #A67C34.
- Tipografi hissi: modern serif (Fraunces). Editoryal, Kinfolk/Aesop estetiği.
- İhtiyaç: iOS uygulama ikonu + logomark. Küçük boyutta (60px) okunur, düz (flat),
  ikon içinde yazı YOK.

---

## Ana prompt (önerdiğim konsept: aynaya bakan iki soru işareti)

İki soru işareti karşılıklı duruyor; aralarındaki negatif boşluk belli belirsiz bir
kalp/elmas formu çiziyor. "Soru soran iki kişi = bir bütün" — ürünün kendisi.

```
Minimalist luxury app icon for a premium couples quiz game. Two mirrored
question marks facing each other, their curves forming a subtle abstract
heart in the negative space between them. Flat vector logomark, bronze
(#A67C34) on warm ivory (#F6F2EA), rounded-square iOS app icon. Elegant
editorial style, refined thin strokes, perfect symmetry, generous
whitespace. Aesop / Kinfolk aesthetic, timeless and understated.
No text, no gradients, no shadows, no 3D, no faces, no clip-art hearts,
not cute, not playful.
```

## Varyant 2 — kesişen iki halka

```
Minimalist luxury app icon: two thin-line circles gently overlapping,
a small solid bronze diamond exactly at their intersection. Flat vector,
bronze (#A67C34) on warm ivory (#F6F2EA), rounded-square iOS app icon.
Editorial, jewelry-like restraint, perfect geometry, generous whitespace.
No text, no gradients, no 3D, no hearts, not cute.
```

## Varyant 3 — serif monogram

```
Elegant custom serif monogram logomark: a single refined letter "D" with
high-contrast strokes (Fraunces / Didot spirit), mirrored by a hairline
ghost reflection of itself facing it. Flat vector, charcoal (#241D13) on
warm ivory (#F6F2EA), rounded-square iOS app icon. Luxury fashion-house
restraint. No gradients, no 3D, no extra text, no ornaments.
```

## Negatif prompt (destekleyen araçlar için)

```
text, letters, words, gradients, 3D, bevel, glossy, shadow, cartoon,
cute, kawaii, cupid, arrow, red heart, pink, clip art, mascot, faces
```

## Kullanım notları

- 1:1 oran, mümkünse "flat vector / logomark" stilinde üret.
- Beğendiğin çıktıyı bana getir — temiz SVG olarak yeniden çizerim
  (uygulama ikonu, favicon ve App Store görseli tek kaynaktan türer).
- Küçük boyut testi: çıktıyı 60px'e küçült; hâlâ ne olduğu anlaşılıyorsa doğru yoldayız.
